# SoulHunter (Web)

Roguelite survival top-down 2D, thể loại kiểu Vampire Survivors với cơ chế **Fusion Upgrade** làm điểm khác biệt. Xây bằng Phaser 3 + TypeScript + Vite.

## Bắt đầu

```bash
npm install
npm run dev
```

Mở http://localhost:5173

## Tài liệu

- [`CLAUDE.md`](./CLAUDE.md) — hướng dẫn cho AI Agent khi làm việc trên repo này (cấu trúc, quy ước code, thứ tự triển khai)
- [`docs/GDD.md`](./docs/GDD.md) — Game Design Document đầy đủ (vũ khí, upgrade, fusion, quái, boss, meta progression...)

## Trạng thái

Đang ở giai đoạn scaffold — xem mục "Trạng thái hiện tại / việc cần làm" trong `CLAUDE.md` để biết thứ tự triển khai đề xuất.
