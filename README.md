# SoulHunter (Web)

Roguelite survival top-down 2D, thể loại kiểu Vampire Survivors với cơ chế **Fusion Upgrade** làm điểm khác biệt. Xây bằng Phaser 3 + TypeScript + Vite.

## Bắt đầu

```bash
npm install
npm run dev
```

Mở http://localhost:5173

## Tài liệu

- [`CLAUDE.md`](./CLAUDE.md) — điểm bắt đầu (entry point): công nghệ, quy tắc bắt buộc, danh sách tài liệu, workflow cho AI Agent
- `docs/architecture/` — kiến trúc kỹ thuật
- `docs/development/` — quy tắc code, trạng thái triển khai, roadmap
- `docs/gameplay/` — thiết kế game (vũ khí, upgrade, fusion, quái, boss, map, meta progression, UI)
- `docs/assets/` — hướng dẫn asset (đồ họa, âm thanh)
